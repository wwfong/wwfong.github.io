**Role:** Research Engineer (Audio)
**Period:** April 2018; January 2019 – September 2019
**Organisation:** AI Music (later acquired by Apple)

- Built, trained, and exposed via API an automatic music tagging system using a CRNN model on STFTs.
- Built and evaluated a preference-optimised music generation system using genetic algorithms and heuristics.
- Built a spectral extractor and a vocal reverb estimation system using a CNN model on Mel spectrograms.

**Stack:** Python, Keras/PyTorch, Librosa, CRNN/CNN architectures, REST APIs.
